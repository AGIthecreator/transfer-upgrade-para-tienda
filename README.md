# 📱 Herramientas de Optimización y Transferencia Móvil

Este repositorio contiene dos scripts de Python diseñados para facilitar la personalización, limpieza y migración de datos entre dispositivos móviles.

---

## 🛠️ Herramientas Disponibles

### 1. ⚙️ Configurador de Dispositivo (`configurador.py`)
Este script permite personalizar el teléfono a gusto del cliente, optimizar el sistema y eliminar aplicaciones antiguas o innecesarias de forma automática.

**Requisitos para el uso:**
*   Conectar el móvil al ordenador mediante **USB**.
*   Tener activada la **Depuración USB** en las opciones de desarrollador del teléfono.
*   **Importante:** La carpeta `media/` debe estar en la misma ubicación que el archivo `configurador.py`.

**Instrucciones:** Una vez conectado el móvil con la depuración activa, ejecuta el script y sigue las opciones de personalización y optimización en pantalla.

---

### 2. 📲 Transferencia de Datos (`transferencia.py`)
Permite migrar todo el contenido multimedia y documentos de un teléfono antiguo a uno nuevo manteniendo la estructura original.

**Requisitos para el uso:**
*   Es **obligatorio** tener **2 puertos USB** disponibles.
*   Conectar ambos teléfonos (antiguo y nuevo) simultáneamente.
*   Tener la **Depuración USB** activa en ambos dispositivos.

**Instrucciones:** Al abrir el script, se presentará la opción para copiar automáticamente todas las imágenes, videos, audios y documentos en el mismo orden al nuevo dispositivo.
